# AdmitAudit.ai

AdmitAudit.ai is an AI-powered admissions essay review tool for international applicants. It focuses on structured feedback, logic optimization, and voice preservation without ghostwriting full essays.

## MVP Goal

A signed-in user pastes an SOP, clicks review, completes PayPal payment, and receives:

- admissions-style scoring
- paragraph-level diagnostics
- AI-likeness risk flags
- rewrite suggestions that preserve the original meaning

## Initial Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Supabase Auth + Postgres
- PayPal
- Claude 3.5 Sonnet through an OpenAI-compatible API
